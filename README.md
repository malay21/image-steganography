
BMESSENGER
> This is a python based image steganography which provides double protection.
> this applicationconverts text into base64 and merge it with image.
---
## Requirement
 - python 3.8
 - pillow (python image library)
    - can be installed via <br/>
    **pip install pillow**
 - tkinter
    - can be installd via
    **pip install tk** <br/>
